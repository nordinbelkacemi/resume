Face Filter

This project was done in a team of 3. The goal was simply to get ourselves aquainted with google's MediaPipe library. We used Python and OpenCV to capture the webcam's video feed, then Mediapipe's neural networks provided us with facial and body landmarks. Using the coordinates of these landmarks, we were able to implement a snapchat-like filter: a clown's nose appearing on the user's face in real time.
